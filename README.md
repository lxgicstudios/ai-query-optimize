Optimize SQL and Prisma queries using AI.

## Install

```bash
npm install -g ai-query-optimize
```

## Usage

```bash
npx ai-query-optimize ./src/queries/
npx ai-query-optimize ./prisma/
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
